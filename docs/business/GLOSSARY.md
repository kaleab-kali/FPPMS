# Glossary

> Terms and definitions used in PPMS

---

## General Terms

| Term | Definition |
|------|------------|
| **PPMS** | Police Personnel Management System |
| **Tenant** | An organization using the system (e.g., Federal Police) |
| **Multi-tenant** | System architecture where multiple organizations share the same application |
| **HQ** | Headquarters - main administrative center |

---

## Employee Terms

| Term | Definition |
|------|------------|
| **Employee ID** | Unique identifier (e.g., FPC-0001/25) |
| **Military Employee** | Uniformed police officer (prefix: FPC) |
| **Civilian Employee** | Non-uniformed staff (prefix: FPCIV) |
| **Temporary Employee** | Contract/temporary worker (prefix: TMP) |
| **Transfer** | Movement of employee between centers |
| **Departure** | Employee leaving the organization |

---

## Organization Terms

| Term | Definition |
|------|------------|
| **Center** | Physical work location/station |
| **Department** | Organizational unit within a center |
| **Position** | Job title/role |
| **Rank** | Military grade (for military employees) |

---

## Military Terms

| Term | Definition |
|------|------------|
| **Enlisted** | Lower ranks (Constable to Sergeant) |
| **NCO** | Non-Commissioned Officer (Staff Sergeant to Chief Warrant Officer) |
| **Officer** | Commissioned ranks (2nd Lieutenant to General) |
| **Salary Step** | Pay grade within a rank (1-8) |

---

## Workflow Terms

| Term | Definition |
|------|------------|
| **CRUD** | Create, Read, Update, Delete operations |
| **Status** | Current state of an entity in its workflow |
| **Approval** | Authorization step in a workflow |
| **Clearance** | Verification/sign-off process |

---

## Disciplinary Terms

| Term | Definition |
|------|------------|
| **Article 30** | Minor disciplinary violation |
| **Article 31** | Serious disciplinary violation (affects rewards) |
| **Investigation** | Formal inquiry into a complaint |
| **Complaint** | Formal accusation against an employee |

---

## Technical Terms

| Term | Definition |
|------|------------|
| **DTO** | Data Transfer Object - defines request/response structure |
| **API** | Application Programming Interface |
| **JWT** | JSON Web Token - authentication token |
| **RBAC** | Role-Based Access Control |
| **ORM** | Object-Relational Mapping (Prisma) |

---

## Access Terms

| Term | Definition |
|------|------------|
| **Permission** | Specific action a user can perform |
| **Role** | Collection of permissions |
| **Access Scope** | Data visibility level (all centers, own center, etc.) |
| **Guard** | Security check in the API |

---

## Calendar Terms

| Term | Definition |
|------|------------|
| **Gregorian** | International calendar system |
| **Ethiopian** | Ethiopian calendar (7-8 years behind Gregorian) |
| **Fiscal Year** | Financial/administrative year |
